# My Web GIS Map — GIS 5220/420

## Project Description
This project is a simple Web GIS application created using Leaflet. It displays city locations in the United States from a GeoJSON file and shows attribute information in popups.

## Data Source
Sample GeoJSON dataset created for the lab assignment.

## Features
- Interactive Leaflet web map
- OpenStreetMap basemap
- GeoJSON point data
- Popups showing city details
- Scale bar
- Deployed on GitHub Pages and Netlify

## Technologies Used
- HTML
- CSS
- JavaScript
- Leaflet
- GeoJSON
- GitHub Pages
- Netlify
